🏠 الرئيسية HTML CSS JavaScript Python

📚 الدرس الأول: ما هي HTML؟

HTML هي اللغة التي نستخدمها لإنشاء هيكل صفحات المواقع.

باستخدام HTML يمكنك إضافة العناوين والفقرات والصور والروابط وغيرها.

📝 الدرس الثاني: إنشاء أول صفحة

هذا مثال بسيط لأول صفحة HTML: 

<!DOCTYPE html>
<html>

<head>
    <title>موقعي الأول</title>
</head>

<body>

    <h1>مرحبًا بك في موقعي</h1>

    <p>أنا أتعلم HTML مع UDH</p>

</body>

</html>

🏷️ الدرس الثالث: العناوين

في HTML يوجد 6 مستويات للعناوين من h1 إلى h6. 

<h1>عنوان كبير</h1>

<h2>عنوان متوسط</h2>

<h3>عنوان أصغر</h3>

✍️ الدرس الرابع: الفقرات

نستخدم الوسم p لكتابة الفقرات. 

<p>
هذا أول نص في موقعي.
</p>

🔗 الدرس الخامس: الروابط

يمكنك إنشاء رابط باستخدام الوسم a. 

<a href="https://example.com">
اضغط هنا
</a>

🎯 تحدي صغير

حاول إنشاء صفحة تحتوي على:

1️⃣ عنوان باسمك
2️⃣ فقرة تعرف فيها نفسك
3️⃣ رابط لموقع تحبه

🚀 حاول تنفيذ التحدي بنفسك!
